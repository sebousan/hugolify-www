## Setup

### Params

{{< alert text="`/config/default/params.yaml`" state="light" >}}

```yml
search:
  enable: false # Set true to enable search
  filter: false # Set true to add filters by type
  sort: false 
```

### Command

Use specific command to build and index 

```bash
yarn build:search
```

### PostCSS

```js
safelist: {
  deep: [/^pagefind-ui/]
}
```
