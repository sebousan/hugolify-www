# Video block

Displays a video element.

## Frontmatter

```yml
blocks:
  - type: video
    heading:
      surtitle: '' // string
      title: '' // string
      text: '' // markdown
    video: {}
    grid: '' // [small, medium, large, full]
    background: false // boolean
```
