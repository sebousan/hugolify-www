# Logos block

Displays a video element.

## Frontmatter

```yml
blocks:
  - type: logos
    heading: {}
    layout: '' // grid (default) or carousel
    items: []
    layout: '' // [grid, carousel]
    carousel: {} // object
    background: false
```
