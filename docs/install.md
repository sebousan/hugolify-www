## Repository clone

Clone with submodules
```
git clone git@github.com:sebousan/hugolify-template.git --recurse-submodules
```

## Package install

Avec yarn
```
yarn
```

## Hugo install

Check the Hugo documentation https://gohugo.io/installation/

In macOS with brew
```
brew install hugo
```
