# Install

## Github template

### Use template

Create your repository with Hugolify template
https://github.com/hugolify/hugolify-template/generate

### Or clone repository

```shell
git clone git@github.com:hugolify/hugolify-template.git
```

## Packages install

With yarn

```shell
yarn
```

With npm

```shell
npm install
```
