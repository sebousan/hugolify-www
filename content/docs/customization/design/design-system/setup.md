{{< alert-block title="hugolify-theme-design-system" state="info" >}}
It is a future design module for **hugolify-theme**. It provides a complete styling layer without any CSS framework dependency, using CSS custom properties (design tokens) and a lightweight icon system.

It is intended as a drop-in replacement for **hugolify-theme-bootstrap**, switching is a one-line config change, no template or content changes required.
{{< /alert-block >}}
