# Logos block

Displays a video element.

## Frontmatter

```yml
blocks:
  - type: logos
    heading: {}
    layout: '' // grid (default) or carousel
    items: []
    layout: '' // grid (default) or carousel
    carousel: {} // object (only if carousel layout)
    background: false
```
